# Tierlist

Tierlist Maker

## Installation

Clone the repository:

```bash
git clone https://github.com/Bubaak/Tierlist.git
cd Tierlist
```

Install dependencies (if any):

```bash
# For Python (example)
pip install -r requirements.txt

# For Node.js (example)
npm install
```

## Usage

Run the application:

```bash
# For Python (example)
python main.py

# For Node.js (example)
npm start
```
